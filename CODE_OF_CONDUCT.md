# Code of Conduct

MCX aims to be a welcoming, inclusive project where people can collaborate on ideas, code, and content around
Minecraft and AI.

## Basic expectations

- Be respectful and considerate in all interactions.
- Focus criticism on ideas, not people.
- No harassment, hate speech, or discriminatory behavior.
- Assume good faith, but be clear when feedback is about behavior that needs to change.

## Scope

This Code of Conduct applies to all project spaces:

- GitHub issues and pull requests
- Discussions in any associated chat or forum
- Any other official project communication channels

## Reporting

If you experience or witness behavior that violates this Code of Conduct, you can:

- Open a private issue or contact the project owner directly on GitHub.
- Provide as much context as you feel comfortable sharing so the situation can be understood and addressed.

The project maintainer(s) will review reports and may take appropriate action, which can include warnings, limiting
participation, or blocking accounts in severe or repeated cases.

This document is intentionally short and will evolve as the project and community grow.
